# Blog Profissional com Flask + IA + Disqus

Este projeto é o blog oficial de meus projetos como desenvolvedor, construído com **Flask**, **Jinja**, **HTML/CSS**, e integrando **Disqus** para comentários e reações. É hospedado via **Railway.app** e serve como parte do meu portfólio profissional.

## ✨ Funcionalidades

- ✅ Visualização de posts com contador real
- ✅ Comentários integrados via Disqus
- ✅ Registro de reações (coração)
- ✅ Galeria de imagens, vídeos e downloads
- ✅ Página inicial paginada
- ✅ Histórico lateral de postagens
- ✅ Totalmente responsivo
- ✅ Preparado para produção no Railway

## 🚀 Deploy

Este projeto está hospedado em produção via [Railway.app](https://railway.app). Configure as variáveis de ambiente no painel:

```
SECRET_KEY=uma-senha-secreta-aqui
DISQUS_PUBLIC_KEY=sua-chave-publica
```
